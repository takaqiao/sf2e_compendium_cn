# SF2e Compendium 中文翻译

本仓库提供 Foundry VTT 的 SF2e 合集包中文翻译模块（基于 Babele）。

## 内容

- 模块清单：module.json
- 注册脚本：register.js
- 合集翻译：compendium/cn/*.json

## 依赖

- Foundry VTT v13+
- sf2e 系统
- babele 模块

## 安装

在 Foundry VTT 中通过模块清单地址安装：

- https://github.com/takaqiao/sf2e_compendium_cn/releases/latest/download/module.json

## 说明

本模块仅提供中文翻译数据，不包含 SF2e 系统本体。
